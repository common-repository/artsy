Art Direction for WordPress - Allows you to enter custom CSS to be applied per post/page.

== Description ==

An update on the incredible Art Direction plugin by NoÃ«l Jackson (http://noel.io) and Dave Rupert (http://daverupert.com).  Allows you to enter custom CSS to be applied per post/page.

Additional features include:

* Optionally place span classes on your post/page title characters 
* Optionally place span classes on your post/page title words
* Ability to responsivley use viewport units on post/page title

== Installation ==

1. Upload `artsy` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Artsy Styles editor under the post/page entry text to enter custom CSS per post/page
